# AI-procted-exam
ProctorAI is an AI-powered online proctoring system designed to ensure academic integrity during remote examinations. This project applies core concepts from Applied Machine Learning and AI Essentials, including real-time data processing, behavioral analytics, and anomaly detection.

The system uses computer vision + machine learning to monitor student behavior through a webcam and detect suspicious activities such as cheating, tab switching, or abnormal head movements.
